# Image Gallery Project

This is a simple responsive image gallery made using HTML, CSS, and JavaScript.

## Features

- Responsive image grid
- Category filter buttons
- Image captions
- Lightbox image preview
- Previous and next image buttons
- Zoom option
- Download image option
- Keyboard controls
- Mobile navigation menu

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

The gallery shows image cards on the page. Users can click any image to open it in a larger popup view called a lightbox.

Inside the lightbox, users can:

- Close the image preview
- Move to the previous image
- Move to the next image
- Zoom the image
- Download the image

The filter buttons allow users to view images by category, such as all images, nature, city, or people.

## Keyboard Controls

- Escape: Close the lightbox
- Left Arrow: Show previous image
- Right Arrow: Show next image

## Project Files

- `index.html` contains the page structure
- `style.css` contains the design and layout
- `script.js` contains the gallery functionality

## Purpose

This project was created to practice JavaScript DOM manipulation, responsive design, image filtering, and lightbox functionality.
